# СКУД КЕЛЕТ

Система контроля и управления доступом на базе контроллеров **ЭРА-500**.  
Полная замена программного обеспечения ЭНТ Контроль Доступа с веб-интерфейсом.

---

## Стек

| Компонент | Технология |
|-----------|-----------|
| Бэкенд | Python 3.10 + FastAPI + uvicorn |
| База данных | SQLite (aiosqlite) |
| Фронтенд | React 18 + Vite + Tailwind CSS |
| Веб-сервер | Nginx (reverse proxy) |
| Протокол контроллеров | UDP 7714/7715 (ЭРА-500) |

---

## Функционал

- 📡 **Live мониторинг** — события прохода в реальном времени через WebSocket
- 👥 **Сотрудники** — база с фото, несколько карт на сотрудника, сканирование карты через считыватель
- 🖥 **Контроллеры** — мониторинг онлайн/оффлайн, управление (открыть/закрыть), добавление по IP
- 🏢 **Структура** — иерархия Компания → Город → Подразделение, привязка контроллеров к подразделениям
- 📅 **Графики** — рабочие графики по дням недели, привязка к подразделениям и сотрудникам
- 👤 **Пользователи** — управление доступом в систему (Администратор / Отдел кадров / Охрана / Просмотр)
- 📊 **Отчёты** — табель по сотруднику, отчёт присутствия на текущий день

## Разработка

### Бэкенд

```bash
cd /opt/skud/backend
/opt/skud/venv/bin/uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

### Фронтенд

```bash
cd /tmp/skud_build
npm run build
bash /opt/skud/restore_photos.sh   # обязательно после каждой сборки!
```

### Перезапуск сервиса

```bash
kill $(pgrep -f uvicorn)
sleep 1
cd /opt/skud/backend
/opt/skud/venv/bin/uvicorn main:app --host 127.0.0.1 --port 8000 --workers 1 &
```

# Восстановить фото
bash /opt/skud/restore_photos.sh

## Протокол ЭРА-500

Контроллеры общаются по UDP на портах **7714** (входящий) и **7715** (исходящий).

### Структура пакета события прохода (32 байта)

| Байты | Значение |
|-------|----------|
| `[0]` | `0x23` — маркер пакета |
| `[4]` | `0x08` — тип: событие прохода |
| `[6:11]` | MAC-адрес контроллера |
| `[12]` | Точка прохода (`0x04`=вход, `0x05`=выход) |
| `[13:17]` | Дата и время |
| `[22:25]` | ID карты (3 байта, big-endian) |
| `[26]` | Счётчик события |
| `[28:30]` | ID записи в памяти контроллера |

### Структура heartbeat пакета (63 байта)

| Байты | Значение |
|-------|----------|
| `[0]` | `0x23` — маркер |
| `[4]` | `0x01` — тип: heartbeat |
| `[6:11]` | MAC-адрес контроллера |

Heartbeat отправляется каждые ~2 секунды. Если пакет не получен более 30 секунд — контроллер считается оффлайн.
